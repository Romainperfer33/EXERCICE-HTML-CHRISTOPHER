LES ELEMENTS DE TYPE BLOCK ET INLINE

ELEMENT DE TYPE BLOCK

Occupe 100% la largeur de l'ecran

genere un retour a la ligne pour l'element suivant

ex: p, h1-h2 etc hr

ELEMENT DE TYPE INLINE

occupe que la larguer dont il a besoin

Ne genere pas de retour à la ligne pour l'élement suivant

Ex: stong em a
